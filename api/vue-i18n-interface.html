# t

# t

Get the translation for a given key as string, searching for plurals.If this method is used in a reactive context, it will re-evaluate once the locale changes.The translated string should contain a | separated value of the singular / plural strings

Signature:

t(key: Path, plural: number, options?: TranslateOptions): string;

Parameters

ParameterTypeDescription
keyPathThe key to get the translation for
pluralnumberWhich plural string to get. 1 returns the first one.
optionsTranslateOptionsAdditional options for translation

# t

Get the translation for a given key as string, searching for plurals.If this method is used in a reactive context, it will re-evaluate once the locale changes.

Signature:

t(key: Path, defaultMsg: string, options?: TranslateOptions): string;

Parameters

ParameterTypeDescription
keyPathThe key to get the translation for
defaultMsgstringThe text to return if no translation was found
optionsTranslateOptionsAdditional options for translation

# t

Get the translation for a given key as string, replacing indexed placeholders.If this method is used in a reactive context, it will re-evaluate once the locale changes.The translated string should contain a 0, 1, … for each placeholder in the list.

Signature:

t(key: Path, list: unknown[], options?: TranslateOptions): string;

Parameters

ParameterTypeDescription
keyPathThe key to get the translation for
listunknown[]The list of placeholders to replace
optionsTranslateOptionsAdditional options for translation

# t

Get the translation for a given key as string, replacing named placeholders.If this method is used in a reactive context, it will re-evaluate once the locale changes.For each placeholder x, the translated string should contain a x token.

Signature:

t(key: Path, named: NamedValue, options?: TranslateOptions): string;

Parameters

ParameterTypeDescription
keyPathThe key to get the translation for
namedNamedValueThe placeholders to replace
optionsTranslateOptionsAdditional options for translation
