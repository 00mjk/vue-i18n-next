# I18nOptions

I18n Options for createI18n

Signature:

export declare type I18nOptions = I18nAdditionalOptions & (ComposerOptions | VueI18nOptions);

# Remarks

I18nOptions is inherited I18nAdditionalOptions, ComposerOptions and VueI18nOptions, so you can specify these options.
